FOR LOOPS

for(var i = 0; i < 10; i++){
                console.log(i);
            };

Loops allow a way to do something repeatedly. They repeat an action
a number of times. A For Loop continues to repeat until a specified
condition is false. There are 3 parts to a for loop: initial expression,
condition and statement.

FUNCTIONS

function printNumbers() {
                    for (let i = 1; i < 11; i++){
                            console.log(i)
                        }
                    }
                    
                    printNumbers();

A function is a group of code which we can call over and over
without having to write the same code multiple times. Functions have to
be called or invoked. Funtions always return a value. Functions are also
always objects.

ARRAYS

var fruits = ['Apple', 'Banana'];

                        console.log(fruits.length);

An array is a list of objects. Arrays are always in brackets. They
also ALWAYS start at zero. This means that the first item in an array
is in position 0, the the next is in 1 and so on. Knowing the postions
allows you to add or remove items using .push .splice or .pop.
